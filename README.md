# سند — Sened

نظام إدارة ومحاسبة ذكي، ثلاثي اللغة (عربي / فرنسي / إنجليزي)، على شكل تطبيق سطح مكتب (Electron) مدعوم بالذكاء الاصطناعي وبوت تلجرام.

## المزايا

- **المبيعات والمشتريات**: فواتير بيع وشراء مع دفع كامل/جزئي/آجل، وتحديث المخزون تلقائياً.
- **المنتجات والعملاء والموردون**: إدارة كاملة مع بحث فوري.
- **الديون**: صفحة مركزية لديون العملاء ومستحقات الموردين مع تحصيل سريع.
- **الموظفون**: سجل رواتب وصرف مباشر يُسجَّل كمصروف تلقائياً.
- **المساهمون وتوزيع الأرباح**: تسجيل المساهمين بنسبهم المئوية، مع تحقق تلقائي من مجموع النسب (100%)، وحساب حصة كل مساهم من صافي الربح، وتتبع المسحوبات والصافي المستحق لكل مساهم.
- **التقارير**: مبيعات شهرية، تكلفة البضاعة المباعة، الربح الإجمالي والصافي، الأكثر مبيعاً.
- **المساعد الذكي**: يجيب عن أسئلة نشاطك التجاري (مبيعات، مخزون، ديون، أرباح المساهمين) عبر Ollama المحلي أو Claude API.
- **بوت تلجرام**: أوامر جاهزة (`/ملخص`, `/منتجات`, `/فواتير`, `/مساهمين`, `/ديون`) وردود بالذكاء الاصطناعي لأي سؤال آخر.
- **الأمان**: شاشة دخول اختيارية وشاشة قفل، مع تشفير كلمة المرور محلياً.
- **المظهر**: تصميم فاخر ذهبي/أخضر زمردي، مع وضعين ليلي ونهاري.
- **نسخ احتياطي**: تصدير واستيراد كامل البيانات بصيغة JSON.

## التشغيل محلياً

```bash
npm install
npm start
```

## بناء نسخة قابلة للتثبيت

```bash
npm run dist
```

يستخدم [electron-builder](https://www.electron.build/) لإنتاج مثبّت NSIS على ويندوز، AppImage على لينكس، وDMG على ماك.

> **مهم — جهاز المدير (الخادم المضيف) يجب أن يستعمل نسخة التثبيت (Setup) وليس المحمولة (Portable):**
> النسخة المحمولة تنفكّ إلى مجلد مؤقت مختلف في كل تشغيل، فيعتبرها جدار حماية ويندوز برنامجاً
> جديداً كل مرة وتنكسر قاعدة السماح المحفوظة، فيُحجب خادم المزامنة (المنفذ 3050) عن بقية
> الأجهزة على الشبكة. نسخة التثبيت تبقى في مسار ثابت فتُحفظ قاعدة الجدار الناري مرة واحدة.

## الذكاء الاصطناعي

- **محلي مجاني**: ثبّت [Ollama](https://ollama.com) وشغّل أي نموذج (مثل `qwen2`)، وسند يتصل به تلقائياً على المنفذ `11434`.
- **اختياري (أقوى)**: أضف مفتاح Claude API من صفحة الإعدادات لاستخدام `claude-sonnet-5`.

## بوت تلجرام

من صفحة "بوت تلجرام" داخل التطبيق: أنشئ بوتاً عبر [@BotFather](https://t.me/BotFather)، الصق الرمز (Token) وشغّله.

## البنية التقنية

```
main.js          عملية Electron الرئيسية: التخزين المحلي، Ollama/Claude، بوت تلجرام، الطباعة
preload.js       جسر آمن بين الواجهة وعملية Electron (contextIsolation)
renderer/        الواجهة: index.html + app.js + styles.css + i18n.js
assets/          الشعار والأيقونات
```

البيانات تُخزَّن محلياً بصيغة JSON في مجلد بيانات المستخدم (`userData`) — لا خوادم خارجية، لا اشتراكات.
